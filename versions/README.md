# Prior versions

When the Eidos standard releases a new **major** version, the outgoing `EIDOS.md` is
preserved here before `EIDOS.md` is edited in place — e.g. `versions/v1.md` holds the
last 1.x standard once 2.0.0 ships.

The current version always lives in the top-level `EIDOS.md`. See `CHANGELOG.md` for
history and migrations.

This folder is empty until the first major bump.
